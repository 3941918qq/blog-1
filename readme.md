## 关于

本博客使用hexo搭建, 主要用于记录开发中遇到的坑, 新东西的学习总结, 以及一些原创文章.
同时也欢迎任何人投稿, 投稿采用pr的方式, 首先fork和clone项目到本地, 在`/source/_posts`创建新的文件并设置符合格式的文件名, 文章中按照其他文章的数据格式设置文章信息, 然后在下方写上文章内容, commit并且提pr给我, 审核通过后即可展示给大家浏览.
